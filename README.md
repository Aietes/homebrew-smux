# Aietes Smux

## How do I install these formulae?

`brew install aietes/smux/<formula>`

Or `brew tap aietes/smux` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aietes/smux"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
